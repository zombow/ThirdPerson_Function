TPS_Function_Project

1. 기획 (간략)

10~20분정도의 전투기획

SouleLike 기반의 전투로 일반 몬스터와 보스몬스터가 존재

일반 몬스터는 간단하게 상대할수있는 몬스터

보스몬스터의경우 패턴이 존재함

1-1. 전투시스템

전투는 기본적으로
HP, 스테미나, 체간(임시)가 존재하며
가드만 하거나 패링에 실패시 체간이 쌓이며 일정치가 넘어갈시 그로기 상태에 돌입되어 큰피해를 입음
반대로 보스의 공격을 패링하거나 특수전투성공시 보스의 채간이 쌓여 그로기 상태로 만들수 있음 (일정시간동안 움직이지못하고 추가데미지를 받음)

특수전투 - 
특수전투의경우 보스의 특정 공격에 패링성공시 돌입하게되며
이때는 보스의 일방적인 공격이 들어오며 방향키 + 패링으로 공격을 막아야함
중간중간 패링불가공격과함께 점프키로 회피하도록 유도하는 식으로 다채로움을 줄수도 있음
이 특수전투가 끝나면 보스는 그로기와함께 추가데미지 타임을 부여함

1-2. 플레이어
GAS를 이용하여 스킬을 부여하고 GamePlayTag를 이용해 State를 관리

1-3. 보스 / 몬스터
